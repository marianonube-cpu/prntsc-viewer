# Visor de Links de Prnt.sc

Esta es una sencilla herramienta web para visualizar imágenes de `prnt.sc` a pantalla completa sin los anuncios y la interfaz de la página original.

## Uso

1.  **Abre la página principal:**
    -   [https://marianonube-cpu.github.io/prntsc-viewer/](https://marianonube-cpu.github.io/prntsc-viewer/)

2.  **Pega tu link de `prnt.sc`:**
    -   En el campo de texto, introduce la URL completa de la imagen que quieres ver (por ejemplo, `https://prnt.sc/ID_DE_IMAGEN_EJEMPLO`).

3.  **Genera el enlace directo:**
    -   Haz clic en el botón "Generar Link".

4.  **Visualiza la imagen:**
    -   Aparecerá un nuevo enlace. Haz clic en él para abrir una nueva pestaña con la imagen a pantalla completa.

## ¿Cómo funciona?

El proyecto consta de dos archivos:

-   `index.html`: La página principal que contiene el generador de links.
-   `prntsc_viewer.html`: La página que recibe un ID de imagen, extrae la URL de la imagen real desde la página de `prnt.sc` y la muestra a pantalla completa.

# Prnt.sc Link Viewer

This is a simple web tool to view `prnt.sc` images in full screen without the ads and interface of the original page.

## How to Use

1.  **Open the main page:**
    -   [https://marianonube-cpu.github.io/prntsc-viewer/](https://marianonube-cpu.github.io/prntsc-viewer/)

2.  **Paste your `prnt.sc` link:**
    -   In the text field, enter the full URL of the image you want to see (e.g., `https://prnt.sc/EXAMPLE_IMAGE_ID`).

3.  **Generate the direct link:**
    -   Click the "Generate Link" button.

4.  **View the image:**
    -   A new link will appear. Click on it to open a new tab with the image in full screen.

## How it works

The project consists of two files:

-   `index.html`: The main page containing the link generator.
-   `prntsc_viewer.html`: The page that receives an image ID, extracts the real image URL from the `prnt.sc` page, and displays it in full screen.
